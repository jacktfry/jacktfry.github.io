Overview

This project refactors my earlier Turtle Graphics scene to improve organization and readability. I added reusable shape functions, modular scene components, and visual details like flags, clouds, and hills.

Code Sample


def draw_flag(t, x, y):
    """Draw a flag on top of a tower"""
    jump_to(t, x, y)
    t.setheading(90)
    t.pendown()
    t.forward(25)
    t.right(90)
    t.fillcolor("red")
    t.begin_fill()
    for _ in range(3):
        t.forward(12)
        t.right(120)
    t.end_fill()
    t.penup()
    t.setheading(0)

This function adds reusable flag drawing on top of each tower using Turtle commands.

Features Included

  • Modular shape functions: rectangle, square, triangle, circle, star
  • Scene elements: moon, stars, castle, flags, clouds, hills
  • Reusable helper functions for drawing and positioning
